# Tetris Game

## Project Description
This project is a modular and optimized implementation of the classic Tetris game, built using Python and the Pygame library. Originally based on a procedural tutorial, the codebase has been completely modified to follow **Object-Oriented Programming (OOP)** principles. By applying **Encapsulation, Inheritance, Polymorphism, and Abstraction**, this version transforms the original script into a professional-grade game engine that is scalable, readable, and highly maintainable.
